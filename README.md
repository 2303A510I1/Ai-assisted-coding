# AI_project

Flask-based AI project with a web UI and ML inference endpoint.

## Setup
1. optional: create & activate venv
   - `python -m venv .venv`
   - `.venv\Scripts\activate`
2. install dependencies
   - `pip install -r requirements.txt`
3. run app
   - `python main.py`
   - open http://127.0.0.1:5000
4. run tests
   - `python -m pytest -q`
